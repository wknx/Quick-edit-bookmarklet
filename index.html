Editor de páginas instantâneo.

Este é um botão que torna editável o texto do site que você estiver navegando. Você pode alterar o conteúdo e quando você pressiona `ESC` no teclado, o modo de edição termina.

Tenha em mente que essas alterações não são permanentes e nem fazem efeito no banco de dados original do site/aplicação. Todas as alterações são apenas estéticas para fins de sanitização de dados confidenciais para facilitar a edição de uma página antes de criar uma captura de tela ou tarefas semelhantes. Recarregar a página após a edição irá desfazer as alterações.

Arraste o botão abaixo até a sua barra de favoritos para salvar a funcionalidade.

Como Usar:

Clique e assista ao tutorial

Ou siga os passos:

- Clique e arraste o botão abaixo para a sua Barra de Favoritos
Se você não tem certeza se sua Barra de Favoritos está visível, aperte CTRL+SHIFT+B e a barra deve aparecer/sumir

- Quando precisar editar alguma página, clique no botão "Edição Rápida" que você acabou de colocar na Barra de Favoritos.

- Para que o Chrome não ache que palavras estrangeiras são palavras digitadas erradas faça o seguinte:
- Abra as configurações do Chrome e na lateral esquerda, clique em "Idiomas".
- Em "Verificação Ortográfica", habilite quais línguas você quer que o Chrome reconheça ao editar texto.
- Reinicie o Chrome e ele não vai mais apontar palavras estrangeiras como palavras erradas.
